This is a richer way to display your hacker news bio. To use it:

  1. Paste "{your username}.at.hn" somewhere into your profile's 'about' section on HN.
  2. Go to https://{username}.at.hn and you should see things. Add "?refresh" to the query to force a refresh.
  3. Use markdown to make things prettier. Submit PRs on Github if you want to change things.

See an example: padolsey.at.hn.
